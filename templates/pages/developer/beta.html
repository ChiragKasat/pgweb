Beta Information

PostgreSQL release candidates pre-release testing versions, not meant for production use. They are feature-frozen, and we release these to the public for testing before our final release.

PostgreSQL 9.0 Beta is over. Please download the release version of 9.0

If you wish to help with alpha-testing version 9.1, please see the Alpha page.

Version 9.1 will go into beta sometime around March 2011.
